# Basific Example App

這是一個展示 `basific` package 功能的 Flutter 應用程式範例。

## 功能

這個範例應用程式展示了：
- 如何使用 `Calculator` 類別
- 基本的 Flutter UI 組件
- 狀態管理
- Material Design 3 主題

## 如何運行

1. 確保您已經安裝了 Flutter SDK
2. 在終端中導航到 `example` 資料夾：
   ```bash
   cd example
   ```
3. 獲取依賴項：
   ```bash
   flutter pub get
   ```
4. 運行應用程式：
   ```bash
   flutter run
   ```

## 應用程式功能

- **數字顯示**：顯示當前數字和加一後的結果
- **加號按鈕**：增加當前數字
- **減號按鈕**：減少當前數字  
- **重設按鈕**：將數字重設為 0
- **浮動操作按鈕**：快速加一

這個範例展示了如何將 `basific` package 整合到實際的 Flutter 應用程式中。
